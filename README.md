# Facebook Ads Performance Analysis

## Project Overview

This project analyzes Facebook advertising campaign data to uncover key factors influencing user engagement and conversions. Using Python and data analytics techniques, the analysis explores relationships between impressions, clicks, ad spend, total conversions, and approved conversions.

## Objectives

* Analyze the distribution of ad clicks and advertising spend.
* Categorize conversion performance into meaningful groups.
* Compare conversion frequency across different categories.
* Perform hypothesis testing to validate business assumptions.
* Identify relationships between advertising metrics using correlation analysis.
* Build regression models to predict approved conversions based on campaign performance metrics.

## Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* SciPy
* Scikit-learn

## Key Analyses Performed

### Exploratory Data Analysis (EDA)

* Dataset inspection and data type validation.
* Distribution analysis of clicks and ad spend.
* Category-wise conversion frequency analysis.

### Hypothesis Testing

* Pearson Correlation Test between ad spend and approved conversions.
* Independent T-Test for gender-based conversion differences.
* ANOVA test to compare conversion performance across age groups.

### Correlation Analysis

* Examined relationships among:

  * Impressions
  * Clicks
  * Ad Spend
  * Total Conversions
  * Approved Conversions

### Regression Analysis

* Linear Regression model to predict approved conversions.
* Evaluation using:

  * R² Score
  * Mean Squared Error (MSE)
  * Root Mean Squared Error (RMSE)

## Key Insights

* Higher ad spend generally leads to increased approved conversions.
* Clicks show a positive relationship with conversion performance.
* Demographic factors such as age and gender may influence campaign outcomes.
* Regression analysis provides a predictive framework for estimating conversion results from advertising metrics.

## Business Impact

The findings can help marketers:

* Optimize advertising budgets.
* Improve targeting strategies.
* Increase conversion rates.
* Make data-driven campaign decisions.
